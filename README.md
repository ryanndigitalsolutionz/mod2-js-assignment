# Array Practise

**So far, we've noted...**
1. ###indexOf### - *an array method which returns the first index of an element or lists the position of the first index an element. An example is...*

2. ###reverse### - *Reverses the order of elements in an array in place, meaning the original array is modified.*

3. ###lastIndexOf### - *Returns the position of the last occurrence of a specified element in an array.*

4. ###.map()### - *Creates a new array by applying a function to each element of the original array, without changing the original. These are short, precise, and perfect for documentation.*

5. ###.sort()### - *used as a standard way to re-order elements within an array in alphabetical order.*

6. ###.join()### - *used to create and return a new string by combining all other elements of an array.*

7. ###.filter()### - *used to create a new array containing only the elements fro an existing array that passes a specific test/condition.*

8. ###.every()### - *tests whether all elements in the array pass the tests; if it returns true/false.*

9. ###.concat()### - *used to merge two or more strings to create a brand new one.*

10. ###.some()### - *tests whether at least one array passes the test; if it returns true/false.*

11. ###.fill()### - *changes all elements in an array to a static value from a start index to an end index.*

12. ###.reduce()### - *this array accumulates values of the elements in an array.*

13. ###.includes()### - *checks if an array contains a specific value and returns true or false.*

14. ###.Array.isArray()### - *a static method that checks if the passed value is an array.*

15. ###.with()### - *returns a new array with the element at a specific index replaced with a new value.*
